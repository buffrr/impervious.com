Impervious beta

Impervious is a privacy and security-focused browser with native DANE support and a decentralized p2p light client. This is a pre-release beta version. Please report issues/feedback on Github or through the Testflight app.

Features

  • Decentralized p2p Handshake light client
  • Zero-trust browser security using DANE/TLSA without any centralized certificate authorities
  • Access names on Ethereum such as Forever domains and ENS
