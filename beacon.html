Beacon web browser

Beacon is a privacy and security-focused browser with the goal of replacing traditional certificate authorities with decentralized peer-to-peer naming systems, DNSSEC, and DNS-Based Authentication of Named Entities (DANE)!


  • Decentralized p2p Handshake light client that syncs block headers in a few seconds!
  • Zero-trust HTTPS/TLS using DANE/TLSA without centralized certificate authorities
  • Access names on Ethereum such as Forever domains and ENS
  • Beacon is open source
